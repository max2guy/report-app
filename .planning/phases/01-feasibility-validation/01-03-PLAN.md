Sandbox test approach:
1. Run unsigned debug build → record IOKit output (already done in Plan 01)
2. Build release binary: `swift build -c release`
3. Create sandbox.entitlements with `com.apple.security.app-sandbox` + `com.apple.security.device.bluetooth`
4. Sign with ad-hoc identity + Hardened Runtime: `codesign --sign - --entitlements sandbox.entitlements --options runtime .build/release/bt-battery-probe`
5. Run sandboxed binary with --iokit: `.build/release/bt-battery-probe --iokit`
6. Compare results: if sandboxed build returns 0 devices but unsigned returned N devices → IOKit is blocked

Expected outcome (HIGH confidence per RESEARCH.md):
- Sandboxed IOKit will return 0 devices or KERN_SUCCESS with empty iterator
- This confirms "Notarization + .dmg direct distribution" (D-09 branch 2)

SandboxProbe.swift design:
- `runSandboxProbe()` function called from main.swift `--sandbox` flag
- Detects if current process is running under sandbox: `sandbox_check(getpid(), nil, SANDBOX_FILTER_NONE)`
- Runs IOKit scan via `probeIOKit()` and prints results + sandbox status
- Output: `[Sandbox] Process sandbox status: sandboxed / not sandboxed`
- Output: `[Sandbox] IOKit results under current environment: N device(s)`
- Output: `[Sandbox] FEAS-02 result: IOKit is <accessible/blocked> under App Sandbox`

    Step 1 — Replace bt-battery-probe/Sources/bt-battery-probe/SandboxProbe.swift:
    ```swift
    import Foundation
    import IOKit

    // sandbox_check is available via libc on macOS
    // Using Darwin module for low-level sandbox check
    #if canImport(Darwin)
    import Darwin

    private func isRunningInSandbox() -> Bool {
        // sandbox_check returns 0 if NOT sandboxed, 1 if sandboxed
        // SANDBOX_FILTER_NONE = 0
        return sandbox_check(getpid(), nil, 0) != 0
    }
    #else
    private func isRunningInSandbox() -> Bool { return false }
    #endif

    func runSandboxProbe() {
        let sandboxed = isRunningInSandbox()
        print("[Sandbox] Process sandbox status: \(sandboxed ? "SANDBOXED" : "not sandboxed (unsigned/development build)")")

        print("[Sandbox] Running IOKit probe under current sandbox environment...")
        let results = probeIOKit()

        if results.isEmpty {
            print("[Sandbox] IOKit returned 0 devices under current environment")
        } else {
            print("[Sandbox] IOKit returned \(results.count) device(s):")
            for r in results {
                if let pct = r.batteryPercent {
                    print("  \(r.product): \(pct)%")
                } else {
                    print("  \(r.product): no battery data")
                }
            }
        }

        if sandboxed {
            if results.isEmpty {
                print("[Sandbox] FEAS-02 result: IOKit is BLOCKED under App Sandbox (0 devices returned)")
                print("[Sandbox] D-09 Decision: Notarization + .dmg direct distribution required (App Store NOT viable for IOKit battery reads)")
            } else {
                print("[Sandbox] FEAS-02 result: IOKit is ACCESSIBLE under App Sandbox (\(results.count) device(s) returned)")
                print("[Sandbox] D-09 Decision: App Store distribution may be viable (verify with Apple review)")
            }
        } else {
            print("[Sandbox] NOTE: This is an UNSIGNED build. Re-run the sandboxed release build to complete FEAS-02.")
            print("[Sandbox] See sandbox test instructions in RESEARCH.md Pattern 4.")
        }
    }
    ```
